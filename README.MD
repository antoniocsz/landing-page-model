# Modelo de Landing Page, usado pela Equipe do VemPlanejar, usando NextJS que consome Blog da NotionAPI.

## OBS
Popular o .env com as informações contidas no .env.example
----------------------------------------------------------------


# Instalar dependências
```
npm install
```

----------------------------------------------------------------
# Executar em dev.
```
npm run dev
```

# Build do projeto
```
npm run build
```

# Executar em prod.
```
npm run start
```
